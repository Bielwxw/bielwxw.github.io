Experience
_________

JavaScript :

JavaScript é uma linguagem de programação de alto nível amplamente utilizada para desenvolvimento web e aplicativos móveis. Ela desempenha um papel fundamental na criação de interações dinâmicas em páginas da web e é uma linguagem de script versátil que permite aos desenvolvedores adicionar funcionalidades avançadas aos sites.

Html e css :

HTML define a estrutura e o conteúdo de uma página da web, enquanto CSS controla a apresentação e o estilo dessa página. Juntos, eles formam a base do desenvolvimento web moderno, permitindo que os desenvolvedores criem páginas atraentes, acessíveis e bem organizadas.

Education

Instituto Federal de Alagoas :

IFAL é uma instituição de ensino renomada que desempenha um papel fundamental na educação e no desenvolvimento da região de Alagoas, oferecendo uma ampla gama de cursos e programas acadêmicos para estudantes de diferentes níveis de ensino e áreas de interesse.

Desenvolvimento de sistemas :

Os estudantes do curso de Desenvolvimento de Sistemas no IFAL podem esperar aprender linguagens de programação, frameworks, metodologias ágeis, e outras habilidades práticas necessárias para projetar, desenvolver e testar sistemas de software. Além disso, o curso frequentemente enfatiza a resolução de problemas, o trabalho em equipe e a colaboração em projetos práticos, preparando assim os alunos para o mercado de trabalho na indústria de tecnologia da informação.
